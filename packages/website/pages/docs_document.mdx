# Document

The `routes/_document.js` file is used to wrap all of your pages with `<html>`, `<head>` and `<body>` tags.

```jsx title='routes/_document.js' lineNumbers
import { css } from 'firebolt'

export default function Document({ children }) {
  return (
    <html lang='en'>
      <head>
        <meta charSet='utf-8' />
        <meta name='viewport' content='width=device-width, initial-scale=1' />
        <style
          global={css`
            :root {
              background: #ffffff;
            }
          `}
        />
      </head>
      <body>{children}</body>
    </html>
  )
}
```

Place things that common to your entire app such as global styles, favicons and fonts.

There are no special framework-specific components here, it's just regular old JSX!
