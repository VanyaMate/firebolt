# Document

The Document component is used to generate the outer shell of your app. All of your pages will be wrapped with this component and it's usually where you'll set up any global styles, meta tags, favicons and fonts that you need.

```jsx title='components/Document.js' lineNumbers
import { css } from 'firebolt'

export function Document({ children }) {
  return (
    <html lang='en'>
      <head>
        <meta charSet='utf-8' />
        <meta name='viewport' content='width=device-width, initial-scale=1' />
        <style
          global={css`
            :root {
              background: #ffffff;
            }
          `}
        />
      </head>
      <body>{children}</body>
    </html>
  )
}
```

You can learn more about how to insert dynamic custom metadata on the [Metadata](/docs/metadata) page.
