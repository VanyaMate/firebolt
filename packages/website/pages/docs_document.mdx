# Document

The Document component is the root of your Firebolt app and includes the `<html>`, `<head>` and `<body>` elements that are common across all pages.

```jsx title='components/Document.js' lineNumbers
import { css } from 'firebolt'

export function Document({ children }) {
  return (
    <html lang='en'>
      <head>
        <meta charSet='utf-8' />
        <meta name='viewport' content='width=device-width, initial-scale=1' />
        <style
          global={css`
            :root {
              background: #ffffff;
            }
          `}
        />
      </head>
      <body>{children}</body>
    </html>
  )
}
```

Use this to place your global styles, meta tags, favicons and fonts etc.

There are no special framework-specific components, it's just regular old JSX!
