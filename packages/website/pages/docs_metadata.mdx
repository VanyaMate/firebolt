# Metadata

In a Firebolt app you can place `<title>` and `<meta>` tags anywhere in your app and they'll automatically be hoisted to the `<head>`.

```jsx title='pages/index.js' lineNumbers
export default function Home() {
  return (
    <main>
      <title>Home</title>
      <meta name='description' content='The home page' />
      {/* ...rest of the page... */}
    </main>
  )
}
```

Use them for setting the browser window title, SEO and Open Graph tags etc.

You can use the `key` prop to categorize and de-duplicate tags of the same kind.
