# Metadata

You can set your page metadata by simply inserting `<title>` and `<meta>` tags anywhere in your app hierarchy.

```jsx title='pages/index.js' lineNumbers
export default function Home() {
  return (
    <main>
      <title>Home</title>
      <meta name='description' content='The home page' />
      {/* ...rest of the page... */}
    </main>
  )
}
```

Tags will automatically be hoisted to the `<head>` of your app.

You can use a `key` prop on these elements if you need to de-duplicate tags of the same kind.
