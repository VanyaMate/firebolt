# Metadata

In a Firebolt app you can simply place `<title>` and `<meta>` tags anywhere in your app hierarchy and they'll automatically be hoisted to the `<head>`.

```jsx title='pages/index.js' lineNumbers
export default function Home() {
  return (
    <main>
      <title>Home</title>
      <meta name='description' content='The home page' />

      {/* page content */}
    </main>
  )
}
```

Use these to set per-page tags for the window title, SEO, open graph and social media preview links.

You can use the `key` prop to de-dupe tags of the same kind.
