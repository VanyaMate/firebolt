# Metadata

To set the browser tab title and insert meta tags for SEO or social sharing functionality you can use `<title>` and `<meta>` tags anywhere in your app hierarchy.

```jsx title='pages/index.js' lineNumbers
export default function Home() {
  return (
    <main>
      <title>Home</title>
      <meta name='description' content='The home page' />
      {/* ...rest of the page... */}
    </main>
  )
}
```

Tags will automatically be hoisted to the `<head>` of your app.

If you need to you can use the `key` prop to deal with tags of the same kind.
